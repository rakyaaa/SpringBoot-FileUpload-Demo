To upload the File.
Run the project as Spring boot app.

Go to Postman: 
Then select the Post Method.
Then Go to Body.
Then go to form-data.
Then select the key as file.
And give the heading as file and select the file.

To get the file details Use : localhost:8100/fileDetails
To upload the file Use : localhost:8100/uploadFile
You will get the string URL in return , take it and open in browser to view the image
